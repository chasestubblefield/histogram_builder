# HistogramBuilder

Relevant to https://github.com/skyray-inc/interview-backend-histogram

## Usage

Requires `ruby` and `bundler`. Install dependencies with:

```bash
bundle install
```

Then run with:

```bash
./bin/histogram_builder test/example.json
step3: (30.0s) ##############################
step4: (15.0s) ###############
step1: (5.0s)  #####
step2: (1.0s)  #
```
